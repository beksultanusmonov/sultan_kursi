Cascading (kaskadlanish)!

! Kaskadlanish deb - bitta elementga bir nechta css qoidalari ko'rsatilgan bo'lsa shulardan bittasini tasir qilishi.

! Kaskadlanish jarayonida qaysi usul ustunvor bo'lsa shu usulni bir xil stillar tasir qiladi qolganlarniki esa tasir qilmaydi.

Ustunvorlik darajasini korib chiqadigon bolsak:

  1. inlini style
    - Eng birinchi o'rinda inline yozilgan stillar ustunvor bo'ladi.

  2. id
    - id esa inline stildan keyingi birinchi orinda turadigon ustunvor usuldir.

  3. class
    - class esa id dan keyingi ustunvor usul.

  4. attribute
    - class dan keyingi usul bu attributedir.

  5. tag
    - eng oxirgisi bu tag yordamida berilgan stil usulidir.

! Css devor qoidasi bilan ishlaydi yani devorga suv sepsangiz tepadan pastga qarab oqib tushdadi, misol uchun bitta elementga ikkita class yordamida color berildi bu holda css eng oxirgi (pastda) yozilgan kodni ishlatadi.

Shorthands (Qisqartmalar)ga kirish!

! Qisqartmalar juda ko'p ishlatilinadi va ular qulaydir misol uchun kengaytmasidagi holatni ko'rsak:

  • .box {

    }

! Ko'rib turganingizdek har bir tomoniga bir xil qiymat berilayapti va har biriga alohida yozilayapti, bu kod kattalashishiga olib keladi va saytni ishlashini sekinlashtiradi. Shunday muammoni yo'q qilish uchun qisqartmalar o'ylab topilgan. Misol uchun shuni qisqartirib yozsak:

  • .box {
    margin: 10px;
    }

! Bu kod qisqa va tushunarli hisoblanadi va manosi jihatdan yuqoridagi bilan bir xil.

! Agar biz har bir tomoniga alohida bermoqchi bo'lsak u holda qoyidagicha yo'l tutamiz:

  • .box {
    margin: 10px 15px 20px 5px;
    }

  • birinchi 10px
    - bu topga berilgan qiymat hisoblanadi.
  • ikkinchi 15px
    - bu rightga berilgan qiymat hisoblanadi.
  • uchinchi 20px
    - bu bottomga berilgan qiymat hisoblanadi.
  • to'rtinchi 5px
    - bu leftga berilgan qiymat hisoblanadi.

    • Shu tariqa soat strelkasi bo'ylab berib kelinadi!

! Bundan tashqari top va bottomga bir xil o'lchov right va left ga bir xil o'lchov qo'ymoqchi bolsak quyidagi usuldan foydalanamiz:

  • .box {
    margin: 5px 10px;
    }

  • birinchi 5px
    - bu top va bottomga berilganqiymat.
  • ikkinchi 10px
    - bu right va leftga berilganqiymat.

! Huddi shu holatda buni paddingga ham qo'llasak bo'ladi:

  • .box {
    padding-top: 10px;
    padding-right: 10px;
    padding-bottom: 10px;
    padding-left: 10px;
    }

    .box {
    padding: 10px 15px 20px 5px;
    }

    .box {
    padding: 5px 10px;
    }

! Endi border uchun ko'radigon bo'lsak, uning yoyilmasi:

  • .box {
    border-width: 5px;
    border-style: solid;
    border-color: red;
    }

  • border-width
    - bu borderning kengligi.
  • border-style
    - bu borderning stili.
  • border-color
    - bu borderning rangi.

! Keling endi uni qisqartmasini ko'radigon bo'lsak:

  • .box {

Width (kenglik) va Height (balandlik) xossalariga kirish!

Width (kenglik)

! Width bu elementning kengligi belgilaydi va bu yordamida har qanday element uchun kenglik yozsak bo'ladi.

  • .box {
    width: 200px;
    }

! Widthning yana ikki xil turi bor birinchisi bu min-width:

  • .box {
    min-width: 100px;
    }

  • min-width
    - bu elementning eng kichik kengligini belgilaydi va element bundan kichik bo'la olmaydi.

! Ikkinchi turi bu max-width:

  • .box {
    max-width: 500px;
    }

  • max-width
    - bu elementning eng katta kengligini belgilaydi va element bundan katta bo'la olmaydi.

Height (balandlik)

! Width bu elementning balandligini belgilaydi va bu yordamida har qanday element uchun balandlik yozsak bo'ladi.

  • .box {
    height: 200px;
    }

! Widthda bo'lganidek heightda ham ikki xil turi bor birinchisi bu min-height:

  • .box {
    min-height: 100px;
    }

  • min-height
    - bu elementning eng kichik balandligini belgilaydi va element bundan kichik bo'la olmaydi.

! Ikkinchi turi bu max-height:

  • .box {
    max-height: 500px;
    }

  • max-height
    - bu elementning eng katta balandligini belgilaydi va element bundan katta bo'la olmaydi.

Texts (tekslar)ga kirish!

  • text-align: start
    - bu tekstlarni boshlanishidan tekislab beradi.
  • text-align: center
    - bu tekstlarni markazga tekislab beradi.
  • text-align: end
    - bu tekstlarni oxiridan tekislab beradi.
  • text-align: right
    - bu tekstlarni o'ng tomondan tekislab beradi.
  • text-align: left
    - bu tekstlarni chap tomondan tekislab beradi.

! Biz undan tashqari elementlarni vertikal ham joylashtira olamiz. Misol uchun bitta parent element bor va uni ichida har xil balandlikga ega elementlar bor, biz ularni hohlasak tepaga hohlasak pastga va hohlasak o'rtadan joylashtira olamiz!

! Bunday joylashtirishda vertical align dan foydalanamiz:

  • vertical-align: top
    - bu contentlarni tepadan tekislab beradi.
  • vertical-align: middle
    - bu contentlarni o'rtadan tekislab beradi.
  • vertical-align: bottom
    - bu contentlarni pastdan tekislab beradi.

Text decorationga kirish!

! Text decoration bu textning stilliga javob beradigon xossa. Uning bir nechta qiymatlari bor:

    (Tagiga chizilgan matn)

  • text-decoration: line-through;

    - bu textni hech qanday stilsiz yozib beradi. (Toza matn) Bu ko'proq a element yani ilova yaratganimizda tagida chiqib qoladigon chiziqni yo'qotish uchun foydalaniladi.

Pseudo class ga kirish!

! Pseudo classlar eng foydali va kerakli css imkoniyatidir, bu yordamida cursor borganda holatini o'zgarishi, bosib turganda qanday holatda tutishini belgilab qoyamiz.

  • .box:hover {
    color: red;
    }

  • Bu elementga sichqonchani olib borganingizda bo'ladigon hodisa.

  • .box:active {
    color: green;
    }

  • Bu elementga bosib turganingizda bo'ladigon hodisa.

Pseudo class ga kirish!

! Pseudo Elementlar elementga oldingi va keyingi qiymatlarini berishda ishlatilinadi asosan hover bo'lganda yoki active.

Biz bitta box ochalik va ichida h1 da qandaydir so'z bo'lsin:

  • .box:hover h1::after {
    content: '';
    top: 10px;
    background-color: blue;
    }

  • Bu elementga sichqonchani olib borganingizda h1 elementi oldin qanday stillga ega bo'lishini belgilaydi.

  • .box:hover h1::before {
    content: '';
    top: 40px;
    background-color: red;
    }

  • Bu elementga sichqonchani olib borganingizda h1 elementi keyin qanday stillga ega bo'lishini belgilaydi.

Position (Joylashuv)ga kirish!

! Htmlda elementlar chapdan o'ngga yoki tepadan pastga joylashib keladi. Misol uchun block element yaratsak tepadan pastga inline element yaratsak chapdan o'ngga qarab joylashadi.

! Har bir elementning 4ta muhum xossasi bor: top, right, bottom va left.

  • Top
    - Bu elementning tepadan joylashishiga javob beradi.
  • Right
    - Bu elementning o'ngdan joylashishiga javob beradi.
  • Bottom
    - Bu elementning pastdan joylashishiga javob beradi.
  • Left
    - Bu elementning chapdan joylashishiga javob beradi.

! Bundan tashqari bularni margin va padding da ham qo'llash mumkun:

  • margin-top
    - Bu elementning tashqari tepadan joylashishiga javob beradi.
  • margin-right
    - Bu elementning tashqari o'ngdan joylashishiga javob beradi.
  • margin-bottom
    - Bu elementning tashqari pastdan joylashishiga javob beradi.
  • margin-left
    - Bu elementning tashqari chapdan joylashishiga javob beradi.

  • padding-top
    - Bu elementning ichkari tepadan joylashishiga javob beradi.
  • padding-right
    - Bu elementning ichkari o'ngdan joylashishiga javob beradi.
  • padding-bottom
    - Bu elementning ichkari pastdan joylashishiga javob beradi.
  • padding-left
    - Bu elementning ichkari chapdan joylashishiga javob beradi.

! Positionga kelsak, Position qaysi joyga nisbatan joylashishiga javob beradi.

Positionning 5ta qiymati mavjud

  • static
    - bu qat'iy qiymati. Odatda HTMLda har bir element shu qiymatda bo'ladi. Bu qiymatda biz top right left va bottom xossalarini ishlata olmaymiz!
  • relative
    - bu o'ziga nisbatan nisbatan joylashishini belgilaydi.
  • absolute
    - bu parent elementga nisbatan nisbatan joylashishini belgilaydi. Agar absolute qiymatli elementning ichidan yana absolute qiymatga ega element bo'lsa shu elementga parent element hisoblanadi!
  • fixed
    - bu o'ziga nisbatan joylashishini belgilay va scroll qilinganda joyidan jilmaydi.
  • sticky
    - bu relative va fixed xossalarini o'z ichiga olgan xossa hisoblanadi.

z-index va overflow ga kirish!

z-index !

! z-index bu elementning boshqa element ustidan ko'rinishi uchun ishlatilinadi. Misol uchun qaysi elementning z-index katta bo'lsa shu element eng ustida joylashadi.

  • .box-1 {
    z-index: 1;
    }

    .box-2 {
    z-index: 2;
    }

Overflow !

! Odatda biz elementlarni joylashtirganimizda ularni parent elementdan tashqarida ko'rinmasligini yoki oshib qolgan qismini surish orqali ko'rinishini istaymiz bunda bizga overflow yordamga keladi.

Odatda 3ta asosiy xossasi bor:

  • overflow
    - bu elementning ichidagi oshgan qismiga javob beradi.

  • .box {
    overflow: hidden;
    }

  • overflow-x
    - bu elementning ichidagi oshgan qismini x o'qi bo'ylab (garizontal) surilishiga javob beradi.

  • .box {
    overflow-x: auto;
    }

  • overflow-y
    - bu elementning ichidagi oshgan qismini y o'qi bo'ylab (vertical) surilishiga javob beradi.

  • .box {
    overflow-x: auto;
    }

! Bizda overflowdan foydalanganimizda chekka va pastdan scroll bar hosil bo'ladi, uni yo'qotish uchun webkit scrollbardan foydalanamiz:

  • ::-webkit-scrollbar {
    width: 0;
    height: 0;
    }
