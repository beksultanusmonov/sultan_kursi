HTMLga kirish

! Html bu buyruqlar toplami bo`lib u yordamida biz kampuyuterga web buyruqlarini berishimiz mumkin!

  1. Deklaratsiya

    !DOCTYPE html
    - ushbu hujjat HTML5 hujjati ekanligini belgilaydi.

  2. Element

    html lang="en"
    - bu htmlning ilidizi hisoblanib odatda uning tilini eng yani ingliz tili qilinadi.

  3. Element

    head
    - bu htmlning boshi, u yerda unga kerak bo'ladigon linklar ulanadi, ismi qoyiladi.

  4. Element

    meta charset="UTF-8"
    - bu kampuyuter va boshqa qurilmalar bilan piksellari togri kelishini taminlaydi.

  5. Element

    meta name="viewport" content="width=device-width, initial-scale=1.0"
    - Bu element ham qurilmalarda birga bir korinishini taminlaydi.

  6. Element

    title
    - Bu html faylning ismi bolib, u yerga nima web sayt haqida ekanligi yoziladi.

  7. Element

    body
    - Bu htmlning tanasi bolib asosan shu joyda loyiha yaratiladi.

  8. Element

    h1
    - Bu sarlavha bolib, mavzularni katta kichik qilib beradi.

  9. Element

    a
    - Bu sahifadan sahifaga otish uchun ishlatilinadi.

Inputga kirish!

! Inputlar foydalanuvchidan malumot olish yoki dinamik tarzda boshqarish va bir qancha hususiyatlarga ega elementdir

Inputlarning 14ta tipi mavjud:

  1. type="text"
    - Bu matn, ism sharif, turarjoy yoki boshqa turli xil matnlar uchun

  2. type="email"
    - Emailga asoslangan input turi, unda asosan foydalanuvchi royxatdan otayotganda emaili tekshirilishi uchun ishlatiladi

  3. type="password"
    - Parollar va maxfiy sozlar uchun mojallangan input turi

  4. type="radio"
    - Bir nechta kannallardan bittasiga ulanish uchun ishlatilinadi

  5. type="color"
    - Ranglar bilan ishlash uchun mojallangan input turi

  6. type="range"
    - Biron funksiyani qiymatini tanlashda ishlatilinadi asosan video yoki musiqalarni davomiyligi yoki ovozini baland pasligi uchun ishlatilinadi

  7. type="checkbox"
    - Bu bajarish yani biron ish bajarilganda bajardim tugmasini bosadi bu huddi shunday tugma

  8. type="file"
    - Foydalanuvchi tomonidan kiritiladigon faylar uchun mojallangan input turi

  9. type="number"
    - Faqat sonlarni qabul qiluvchi input turi

  10. type="tel"
    - Telefon nomerlar uchun mojallangan input turi

  11. type="search"
    - Malumotlar iz;ash uchun mojallangan input turi

  12. type="time"
    - Soat uchun mojallangan input turi

  13. type="hidden"
    - korinmas input turi

  14. type="button"
    - Jonatish tugmasiga mojalangan input turi

! Inputning eng krakli atributilari:

  • required
    - Agar malumot kiritilmagan bolsa, kiritishni talab qiladi.

  • readonly
    - Inputga hech qanday malumot kirita olmaydi, faqat malumotni korish uchun ishlatilionadi.

Cssga kirish!

! Cssning kengaytmasi Cascading Style Sheets bunig manosi Kaskadlangan Stillar jadvali.

! Css ning versiyalari:

  1. Css-1
    - 1996-yil, dekabr oyi o'rtalarida ishlab chiqilgan.

  2. Css-2
    - 1998-yil, may oyida ishlab chiqilgan. 2-versiya 1-versiya ustida qurilgan bo'lib bazibir imkoniyatlar qoshilgan.

  3. Css-3
    - 1999-yil, iyun oyida ishlab chiqilgan. 3-versiyada koplab funksiyalar qoshilgan holda ishlab chiqilgan va shundan beri css 3 dan foydalanib keliniladi.

Cssni html hujjatga ulash!

! Buning uchun html hujjatga otamiz va head qismida link orqali css faylga yo'l ko'rsatamiz va css fayl ulanadi.

Css turlari!

! Html darsida aytib otilgandek css turlari uch turga bo'linadi:

  1. Inline
    - Elementning style atributida beriladigon stillar

  2. Internal
    - htmlning head qismidagi style bolib u yerda ham stillar yozsak boladi

  3. External
    - Eng yaxshi va eng kop ishlatilinadigon stil turi htmlning head qismida link orqali ulanib boshqa style faylga yozilgan stillar

Inline turi!

Inline turida stil berish uchun elementga style attribute beriladi va qoshtirnoq ichida bir qatorlik css kod yoziladi.

Internal turi!

Internal turida stil berish uchun head elementi ichida style elementi ochiladi va shu joyda css kod yoziladi.

External turi!

External turida stil berish uchun head elementi ichida link ochilib, src da css faylga yo'l korsatiladi va css faylda kod yoziladi.

Selectors (Tanlab oluvchilar)!

Selector turlari:

  • tagName
    - Bu htmlda yozilgan elementlar bolib ular (div, span, h1, p, section va h.k) teg nomi yordamida olinadi. (div)

  • class
    - Bu htmlda yozilgan class bolib u (.) nuqta yordamida olinadi. (.class)

  • id
    - Bu htmlda yozilgan id bolib u (#) reshotka yordamida olinadi. (#id)

  • attribute
    - Bu htmlda yozilgan attribute bolib u ([]) tortburchak qavs yordamida olinadi. ([input="text"])

  • universal
    - Bu htmlning hamma elementiga tenglanadigan selector bolib u (*) yulduzcha yordamida olinadi. (*)
