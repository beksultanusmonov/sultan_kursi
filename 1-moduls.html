HTMLga kirish

! Html bu buyruqlar toplami bo`lib u yordamida biz kampuyuterga web buyruqlarini berishimiz mumkin!

  1. Deklaratsiya

    !DOCTYPE html
    - ushbu hujjat HTML5 hujjati ekanligini belgilaydi.

  2. Element

    html lang="en"
    - bu htmlning ilidizi hisoblanib odatda uning tilini eng yani ingliz tili qilinadi.

  3. Element

    head
    - bu htmlning boshi, u yerda unga kerak bo'ladigon linklar ulanadi, ismi qoyiladi.

  4. Element

    meta charset="UTF-8"
    - bu kampuyuter va boshqa qurilmalar bilan piksellari togri kelishini taminlaydi.

  5. Element

    meta name="viewport" content="width=device-width, initial-scale=1.0"
    - Bu element ham qurilmalarda birga bir korinishini taminlaydi.

  6. Element

    title
    - Bu html faylning ismi bolib, u yerga nima web sayt haqida ekanligi yoziladi.

  7. Element

    body
    - Bu htmlning tanasi bolib asosan shu joyda loyiha yaratiladi.

  8. Element

    h1
    - Bu sarlavha bolib, mavzularni katta kichik qilib beradi.
